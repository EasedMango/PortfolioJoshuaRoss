Nuit Blanche 2022 - Together, Apart

An​ immersive digital art projection invited audiences to witness stories about migration, inclusion, distance, and acceptance, demonstrating how people emerge from hardship – sometimes stronger than ever before.

Project was made during Humber Galleries Nuit Blanche Fellowship, over the span of 14 weeks we created Together, Apart, an immersive and interactive digital art experience.

The main software used for the visuals is a program called TouchDesigner, a node based visual programming language for real time interactive multimedia content. And for audio we used FMOD, a sound effects engine and authoring tool. I also creted a program to automate comunication between FMOD and Touchdesigner using OSC(Open Sound Control) 

Software and Tech Used: TouchDesigner, FMOD, OSC(Open Sound Control), Python, C#, Projection Mapping, Kinects/Depth Camera

Squeaky Engine

My own Game Engine built from the ground up 


I built this engine as a learning experince and a passion project. It incorperates a component system, actor manager, engine gui capabilites, and much more. The graphics api I'm using is OpenGL with GLFW to handle windows, context, and input. The engine also incorperates FMOD for dynamic audio.

I used my engine for my Midstone Project, Lunar, and will continue to use it in the future 


Software and Tech Used: C++, OpenGL, GLFW, GLAD, IMGUI, FMOD

Nuit Blanche 2021 - ​A Way Closer

A​ ​10-minute interactive digital art piece invites you to take on the role of an unnamed character trying to escape the commotions of everyday life.

After my 2022 Fellowship ended I was brought onto help clean up the technical side of the 2021 Fellowship project, A Way Closer.

While not part of the ideation and creation of the proccesss I worked on the experience much later to help make it much more playable. The piece was built in Unity and as I was already familar with it I was able to fix up a lot of bugs and make the experince smoother 

Software and Tech Used: Unity, C#, FMOD
